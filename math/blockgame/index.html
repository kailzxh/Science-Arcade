How to play?

The goal is to destroy the correct boxes by choosing the right box with the correct number or operation (+, -, ร, รท) to match the target output.

Each round presents a target number and a set of possible numbers and operators.

Select the right combination of numbers and operators to match the target.

If you hit the correct box, you destroy other boxes matching the target expression.

If you hit a wrong box, the game is over.

There is no time limit for solving each challenge.

The game continues until you make a wrong answer.

You can restart the game after a wrong answer.
